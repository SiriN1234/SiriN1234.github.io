Windows에 Spark 설치하기

Posted by Hanjeongin on 2022-04-18
#Settings

Java 설치

  • https://www.oracle.com/java/technologies/javase/javase8u211-later-archive-downloads.html

  • 경로변경을 해주어야 함 (Program Files에 띄어쓰기가 있기 때문)

  • jre도 경로 변경

  • 설치 완료

Spark 설치

  • https://spark.apache.org/downloads.html

  • 필요한 버전 찾고 3번 클릭

  • https://www.apache.org/dyn/closer.lua/spark/spark-3.2.0/spark-3.2.0-bin-hadoop3.2.tgz

  • 여기선 3.2.0버전을 쓴다

  • Extract to “spark~” 클릭

  • 폴더명을 spark로 바꾸고 c로 위치 변경

  • conf에 들어가서 log4j.properties.template 파일을 메모장으로 실행

  • 원본은 주석처리 해주고 INFO를 ERROR로 변경하고 저장

WinRAR 설치

  • https://www.rarlab.com/download.htm

  • 그냥 설치하면 됨

winutils 설치

  • https://github.com/cdarlint/winutils

  • 본인에 맞는 버전 들어가서 winutils 설치

  • C드라이브에 winutils 폴더 생성 후 그 안에 bin 폴더 생성

  • bin에 winutils 옮김

  • C드라이브에 tmp 폴더 생성 후 그 안에 hive 폴더 생성

  • cmd를 관리자 권한으로 실행

  • 다음 코드를 입력

cd c:\winutils\bin
winutils.exe chmod 777 \tmp\hive

환경변수 설정

  • 시스템 환경 변수 편집 실행

  • 환경변수 클릭

  • 새로 만들기 클릭

    • 변수 이름 - SPARK_HOME
    • 변수 값 - C:\spark (spark 폴더 경로)

  • 새로 만들기 클릭

    • 변수 이름 - JAVA_HOME
    • 변수 값 - C:\jdk (jdk 폴더 경로)

  • 새로 만들기 클릭

    • 변수 이름 - HADOOP_HOME
    • 변수 값 - C:\winutils (winutils 폴더 경로)

  • Path 들어가서 아래 코드 입력

    • %SPARK_HOME%\bin

    • %JAVA_HOME%\bin

  • 새로 만들기 클릭

    • 변수 이름 - PYSPARK_PYTHON
    • 변수 값 - python

스파크 테스트

  • cmd에서 c\spark 경로에 가서 pyspark 입력

  • 다음 코드 입력

rd = sc.textFile("README.md")
rd.count()
